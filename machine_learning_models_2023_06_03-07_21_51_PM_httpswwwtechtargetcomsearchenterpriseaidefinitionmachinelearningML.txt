<section class="section main-article-chapter" data-menu-title="What is machine learning?">
 <h3 class="section-title"><i class="icon" data-icon="1"></i>What is machine learning?</h3>
 <p>Machine learning (ML) is a type of artificial intelligence (<a href="https://www.techtarget.com/searchenterpriseai/definition/AI-Artificial-Intelligence">AI</a>) that allows software applications to become more accurate at predicting outcomes without being explicitly programmed to do so. Machine learning <a href="https://www.techtarget.com/whatis/definition/algorithm">algorithms</a> use historical data as input to predict new output values.</p>
 <p><a href="https://www.techtarget.com/whatis/definition/recommendation-engine">Recommendation engines</a>&nbsp;are a common use case for machine learning. Other popular uses include fraud detection, spam filtering, malware threat detection, <a href="https://www.techtarget.com/searchcio/definition/business-process-automation">business process automation</a> (BPA) and Predictive maintenance.</p>
</section>   
<section class="section main-article-chapter" data-menu-title="Why is machine learning important?">
 <h3 class="section-title"><i class="icon" data-icon="1"></i>Why is machine learning important?</h3>
 <p>Machine learning is important because it gives enterprises a view of trends in customer behavior and business operational patterns, as well as supports the development of new products. Many of today's leading companies, such as Facebook, Google and Uber, make machine learning a central part of their operations. Machine learning has become a significant competitive differentiator for many companies.</p>
</section>  
<section class="section main-article-chapter" data-menu-title="What are the different types of machine learning?">
 <h3 class="section-title"><i class="icon" data-icon="1"></i>What are the different types of machine learning?</h3>
 <p>Classical machine learning is often categorized by how an algorithm learns to become more accurate in its predictions. There are four basic approaches:<a href="https://www.techtarget.com/searchenterpriseai/definition/supervised-learning">supervised</a> learning, <a href="https://www.techtarget.com/whatis/definition/unsupervised-learning">unsupervised</a> learning, semi-supervised learning and reinforcement learning. The type of algorithm data scientists choose to use depends on what type of data they want to predict.</p>

 <ul class="default-list"> 
  <li><strong>Supervised learning:</strong> In this type of machine learning, <a href="https://www.techtarget.com/searchenterpriseai/definition/data-scientist">data scientists</a> supply algorithms with labeled training data and define the variables they want the algorithm to assess for correlations. Both the input and the output of the algorithm is specified.</li> 
  <li><strong>Unsupervised learning:</strong> This type of machine learning involves algorithms that train on unlabeled data. The algorithm scans through data sets looking for any meaningful connection. The data that algorithms train on as well as the predictions or recommendations they output are predetermined.</li> 
  <li><strong>Semi-supervised learning:</strong> This approach to machine learning involves a mix of the two preceding types. Data scientists may feed an algorithm mostly labeled <a href="https://www.techtarget.com/searchenterpriseai/feature/Using-small-data-sets-for-machine-learning-models-sees-growth">training data</a>, but the model is free to explore the data on its own and develop its own understanding of the data set.</li> 
  <li><strong>Reinforcement learning: </strong>Data scientists typically use <a href="https://www.techtarget.com/searchenterpriseai/definition/reinforcement-learning">reinforcement learning</a> to teach a machine to complete a multi-step process for which there are clearly defined rules. Data scientists program an algorithm to complete a task and give it positive or negative cues as it works out how to complete a task. But for the most part, the algorithm decides on its own what steps to take along the way.</li> 
 </ul>
</section>   
<section class="section main-article-chapter" data-menu-title="How does supervised machine learning work?">
 <h3 class="section-title"><i class="icon" data-icon="1"></i>How does supervised machine learning work?</h3>
 <p>Supervised machine learning requires the <a href="https://www.techtarget.com/searchbusinessanalytics/feature/Key-differences-of-a-data-scientist-vs-data-engineer">data scientist</a> to train the algorithm with both labeled inputs and desired outputs. Supervised learning algorithms are good for the following tasks:</p>
 <ul class="default-list"> 
  <li><strong>Binary classification: </strong>Dividing data into two categories.</li> 
  <li><strong>Multi-class classification: </strong>Choosing between more than two types of answers.</li> 
  <li><strong>Regression modeling:</strong> Predicting continuous values.</li> 
  <li><strong>Ensembling:</strong> Combining the predictions of multiple machine learning models to produce an accurate prediction.</li> 
 </ul>
</section>   
<section class="section main-article-chapter" data-menu-title="How does unsupervised machine learning work?">
 <h3 class="section-title"><i class="icon" data-icon="1"></i>How does unsupervised machine learning work?</h3>
 <p>Unsupervised machine learning algorithms do not require data to be labeled. They sift through unlabeled data to look for patterns that can be used to group data points into subsets. Most types of deep learning, including <a href="https://www.techtarget.com/searchenterpriseai/definition/neural-network">neural networks</a>, are unsupervised algorithms.&nbsp;Unsupervised learning algorithms are good for the following tasks:</p>
 <ul class="default-list"> 
  <li><strong>Clustering:</strong> Splitting the dataset into groups based on similarity.</li> 
  <li><strong>Anomaly detection:</strong> Identifying unusual data points in a data set.</li> 
  <li><strong>Association mining:</strong> Identifying sets of items in a data set that frequently occur together.</li> 
  <li><strong>Dimensionality reduction: </strong>Reducing the number of variables in a data set.</li> 
 </ul>

<section class="section main-article-chapter" data-menu-title="How does semi-supervised learning work?">
 <h3 class="section-title"><i class="icon" data-icon="1"></i>How does semi-supervised learning work?</h3>
 <p>Semi-supervised learning works by data scientists feeding a small amount of <a href="https://www.techtarget.com/searchenterpriseai/feature/Labeled-data-brings-machine-learning-applications-to-life">labeled training data</a> to an algorithm. From this, the algorithm learns the dimensions of the data set, which it can then apply to new, unlabeled data. The performance of algorithms typically improves when they train on labeled data sets. But labeling data can be time consuming and expensive. Semi-supervised learning strikes a middle ground between the performance of supervised learning and the efficiency of unsupervised learning. Some areas where semi-supervised learning is used include:</p>
 <ul class="default-list"> 
  <li><strong>Machine translation:</strong> Teaching algorithms to translate language based on less than a full dictionary of words.</li> 
  <li><strong>Fraud detection:</strong> Identifying cases of fraud when you only have a few positive examples.</li> 
  <li><strong>Labelling data:</strong> Algorithms trained on small data sets can learn to <a href="https://www.techtarget.com/whatis/definition/data-labeling">apply data labels</a> to larger sets automatically.</li> 
 </ul>
</section>   
<section class="section main-article-chapter" data-menu-title="How does reinforcement learning work?">
 <h3 class="section-title"><i class="icon" data-icon="1"></i>How does reinforcement learning work?</h3>
 <p>Reinforcement learning works by <a href="https://www.techtarget.com/searchenterpriseai/feature/5-types-of-machine-learning-algorithms-you-should-know">programming an algorithm</a> with a distinct goal and a prescribed set of rules for accomplishing that goal. Data scientists also program the algorithm to seek positive rewards -- which it receives when it performs an action that is beneficial toward the ultimate goal -- and avoid punishments -- which it receives when it performs an action that gets it farther away from its ultimate goal. Reinforcement learning is often used in areas such as:</p>
 <ul class="default-list"> 
  <li><strong>Robotics:</strong> Robots can learn to perform tasks the physical world using this technique.</li> 
  <li><strong>Video gameplay:</strong> Reinforcement learning has been used to teach bots to play a number of video games.</li> 
  <li><strong>Resource management:</strong> Given finite resources and a defined goal, reinforcement learning can help enterprises plan out how to allocate resources.</li> 
 </ul>

<section class="section main-article-chapter" data-menu-title="Who's using machine learning and what's it used for?">
 <h3 class="section-title"><i class="icon" data-icon="1"></i>Who's using machine learning and what's it used for?</h3>
 <p>Today, machine learning is used in a wide range of applications. Perhaps one of the most well-known examples of machine learning in action is the <a href="https://www.techtarget.com/whatis/definition/recommendation-engine">recommendation engine</a> that powers Facebook's news feed.</p>
 <p>Facebook uses machine learning to personalize how each member's feed is delivered. If a member frequently stops to read a particular group's posts, the recommendation engine will start to show more of that group's activity earlier in the feed.</p>
 <p>Behind the scenes, the engine is attempting to reinforce known patterns in the member's online behavior. Should the member change patterns and fail to read posts from that group in the coming weeks, the news feed will adjust accordingly.</p>
 <p>In addition to recommendation engines, other uses for machine learning include the following:</p>
 <ul class="default-list"> 
  <li><strong>Customer relationship management.</strong> <a href="https://www.techtarget.com/searchcustomerexperience/definition/CRM-customer-relationship-management">CRM software</a> can use machine learning models to analyze email and prompt sales team members to respond to the most important messages first. More advanced systems can even recommend potentially effective responses.</li> 
  <li><strong>Business intelligence.</strong> <a href="https://www.techtarget.com/searchbusinessanalytics/definition/business-intelligence-BI">BI and analytics</a> vendors use machine learning in their software to identify potentially important data points, patterns of data points and anomalies.</li> 
  <li><strong>Human resource information systems.</strong> <a href="https://www.techtarget.com/searchhrsoftware/definition/HRIS">HRIS systems</a> can use machine learning models to filter through applications and identify the best candidates for an open position.</li> 
  <li><strong>Self-driving cars.</strong> Machine learning algorithms can even make it possible for a <a href="https://www.techtarget.com/searchenterpriseai/definition/driverless-car">semi-autonomous car</a> to recognize a partially visible object and alert the driver.</li> 
  <li><strong>Virtual assistants.</strong> <a href="https://www.techtarget.com/searchcustomerexperience/definition/virtual-assistant-AI-assistant">Smart assistants</a> typically combine supervised and unsupervised machine learning models to interpret natural speech and supply context.</li> 
 </ul>
</section>      
<section class="section main-article-chapter" data-menu-title="What are the advantages and disadvantages of machine learning?">
 <h3 class="section-title"><i class="icon" data-icon="1"></i>What are the advantages and disadvantages of machine learning?</h3>
 <p>Machine learning has seen use cases ranging from predicting customer behavior to forming the operating system for self-driving cars.</p>
 <p>When it comes to advantages, machine learning can help enterprises understand their customers at a deeper level. By collecting customer data and correlating it with behaviors over time, machine learning algorithms can learn associations and help teams tailor product development and marketing initiatives to customer demand.</p>
 <p>Some companies use machine learning as a primary driver in their business models. Uber, for example, uses algorithms to match drivers with riders. Google uses machine learning to surface the ride advertisements in searches.</p>
 <p>But machine learning comes with disadvantages. First and foremost, it can be expensive. Machine learning projects are typically driven by data scientists, who command high salaries. These projects also require software infrastructure that can be expensive.</p>
 <p>There is also the problem of machine learning bias. Algorithms trained on data sets that exclude certain populations or contain errors can lead to inaccurate models of the world that, at best, fail and, at worst, are discriminatory. When an enterprise bases core business processes on biased models it can run into regulatory and reputational harm.</p>
</section>      
<section class="section main-article-chapter" data-menu-title="How to choose the right machine learning model">
 <h3 class="section-title"><i class="icon" data-icon="1"></i>How to choose the right machine learning model</h3>
 <p>The process of choosing the right machine learning model to solve a problem can be time consuming if not approached strategically.</p>
 <p><strong>Step 1:</strong> Align the problem with potential data inputs that should be considered for the solution. This step requires help from data scientists and experts who have a deep understanding of the problem.</p>
 <p><strong>Step 2:</strong> Collect data, format it and label the data if necessary. This step is typically led by data scientists, with help from data wranglers.</p>
 <p><strong>Step 3:</strong> Chose which algorithm(s) to use and test to see how well they perform. This step is usually carried out by data scientists.</p>
 <p><strong>Step 4:</strong> Continue to fine tune outputs until they reach an acceptable level of accuracy. This step is usually carried out by data scientists with feedback from experts who have a deep understanding of the problem.</p>
</section>      
<section class="section main-article-chapter" data-menu-title="Importance of human interpretable machine learning">
 <h3 class="section-title"><i class="icon" data-icon="1"></i>Importance of human interpretable machine learning</h3>
 <p>Explaining how a specific ML model works can be challenging when the model is complex. There are some vertical industries where data scientists have to use simple machine learning models because it's important for the business to explain how every decision was made. This is especially true in industries with heavy <a href="https://searchcompliance.techtarget.com/definition/compliance-burden">compliance burdens</a> such as banking and insurance.</p>
 <p>Complex models can produce accurate predictions, but explaining to a lay person how an output was determined can be difficult.&nbsp;</p>

<section class="section main-article-chapter" data-menu-title="What is the future of machine learning?">
 <h3 class="section-title"><i class="icon" data-icon="1"></i>What is the future of machine learning?</h3>
 <p>While machine learning algorithms have been around for decades, they've attained new popularity as <a href="https://www.techtarget.com/searchenterpriseai/definition/AI-Artificial-Intelligence">artificial intelligence</a> has grown in prominence. Deep learning models, in particular, power today's most advanced AI applications.</p>
 <p>Machine learning platforms are among enterprise technology's most competitive realms, with most major vendors, including Amazon, Google, Microsoft, IBM and others, racing to sign customers up for platform services that cover the spectrum of machine learning activities, including data collection, <a href="https://www.techtarget.com/searchbusinessanalytics/definition/data-preparation">data preparation</a>, data classification, model building, training and application deployment.</p>
 <p>As machine learning continues to increase in importance to business operations and AI becomes more practical in enterprise settings, the machine learning platform wars will only intensify.</p>
 <p>Continued research into deep learning and AI is increasingly focused on developing more general applications. Today's AI models require extensive training in order to produce an algorithm that is highly optimized to perform one task. But some researchers are exploring ways to make models more flexible and are seeking techniques that allow a machine to apply context learned from one task to future, different tasks.</p>

<section class="section main-article-chapter" data-menu-title="How has machine learning evolved?">
 <h3 class="section-title"><i class="icon" data-icon="1"></i>How has machine learning evolved?</h3>
 <p>1642 - Blaise Pascal invents a mechanical machine that can add, subtract, multiply and divide.</p>
 <p>1679 - Gottfried Wilhelm Leibniz devises the system of <a href="https://www.techtarget.com/whatis/definition/binary">binary</a> code.</p>
 <p>1834 - Charles Babbage conceives the idea for a general all-purpose device that could be programmed with punched cards.</p>
 <p>1842 - <a href="https://www.techtarget.com/whatis/definition/Augusta-Ada-King-countess-of-Lovelace-Ada-Lovelace-Lady-Byron">Ada Lovelace</a> describes a sequence of operations for solving mathematical problems using Charles Babbage's theoretical <a href="https://www.techtarget.com/whatis/reference/History-of-the-punch-card">punch-card machine</a> and becomes the first programmer.</p>
 <p>1847 - George Boole creates <a href="https://www.techtarget.com/whatis/definition/Boolean">Boolean</a> logic, a form of algebra in which all values can be reduced to the binary values of true or false.</p>
 <p>1936 - English logician and <a href="https://www.techtarget.com/searchenterpriseai/definition/Turing-test">cryptanalyst Alan Turing</a> proposes a universal machine that could decipher and execute a set of instructions. His published proof is considered the basis of computer science.</p>
 <p>1952 - Arthur Samuel creates a program to help an IBM computer get better at checkers the more it plays.</p>
 <p>1959 - MADALINE becomes the first <a href="https://www.techtarget.com/searchenterpriseai/definition/neural-network">artificial neural network</a> applied to a real-world problem: removing echoes from phone lines.</p>
 <p>1985 - Terry Sejnowski's and Charles Rosenberg's artificial neural network taught itself how to correctly pronounce 20,000 words in one week.</p>
 <p>1997 - IBM's Deep Blue beat chess grandmaster Garry Kasparov.</p>
 <p>1999 - A <a href="https://www.techtarget.com/whatis/definition/CAD-computer-aided-design">CAD</a> prototype intelligent workstation reviewed 22,000 mammograms and detected cancer 52% more accurately than radiologists did.</p>
 <p>2006 - Computer scientist Geoffrey Hinton invents the term <a href="https://www.techtarget.com/searchenterpriseai/definition/deep-learning-deep-neural-network">deep learning</a> to describe neural net research.</p>
 <p>2012 - An unsupervised neural network created by Google learned to recognize cats in YouTube videos with 74.8% accuracy.</p>
 <p>2014 - A <a href="https://www.techtarget.com/searchcustomerexperience/definition/chatbot">chatbot</a> passes the Turing Test by convincing 33% of human judges that it was a Ukrainian teen named Eugene Goostman.</p>
 <p>2014 - Google's AlphaGo defeats the human champion in Go, the most difficult board game in the world.</p>
 <p>2016 - LipNet, DeepMind's artificial intelligence system, identifies lip-read words in video with an accuracy of 93.4%.</p>
 <p>2019 - Amazon controls 70% of the market share for <a href="https://www.techtarget.com/searchcustomerexperience/definition/virtual-assistant-AI-assistant">virtual assistants</a> in the U.S.</p>
